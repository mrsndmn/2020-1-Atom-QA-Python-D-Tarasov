 
## nginx-log-parser.py

Скрипт на питончике.

Аргументы:
* `--dir` директория, в которой будем искать access.log
* `--out` файл, в который будем записывать результат

## nginx-log-parser.sh

Скрипт на баше

* `-d` директория, в которой будем искать access.log
* `-o` файл, в который будем записывать результат
